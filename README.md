Projek ini adalah aplikasi web sederhana yang dibangun menggunakan Laravel untuk memenuhi tugas learning labs BackEnd. Aplikasi ini berfungsi sebagai sistem manajemen inventaris aset kantor.

Aplikasi ini memiliki fitur **CRUD (Create, Read, Update, Delete)** penuh untuk mengelola data aset, serta fitur **Pencarian** yang fungsional untuk memfilter data berdasarkan nama, kategori, atau lokasi.

### Alasan Memilih Tema

Tema **"Inventaris Aset"** ini dipilih karena sangat ideal untuk mendemonstrasikan fungsionalitas CRUD dan Pencarian secara profesional dan *real-world*.

Tema ini mencerminkan kasus penggunaan di dunia nyata (seperti aplikasi SaaS) di mana data perlu dikelola, dicari, dan ditampilkan dengan jelas dalam sebuah dashboard. Desainnya dibuat "apik" dan modern dengan *sidebar* gelap dan area konten terang untuk memanjakan mata pengguna.

### Tumpukan Teknologi (Stack)

* **Framework:** Laravel 11
* **Database:** MySQL
* **Tampilan (UI):** Tailwind CSS & Alpine.js (dimuat via CDN untuk kesederhanaan dan anti-error).
